# Telco Customer Churn Analysis

## 1. Problem Definition
Customer churn is a critical business issue for telecom companies, as acquiring new customers typically costs more than retaining exiting ones.

## 2. Goal of Analysis
The goal of this analysis is to identify key factors associated with customer churn and build an interpretable model to support retention decisions.

## 3. Dataset Overview
- Dataset: Telco Customer Churn
- Target variable: Churn (Yes / No)
- Features include customer demographics, service usage, and contract information.

## 4. Analysis Approach
1. Exploratory Data Analysis (EDA)
2. Feature preprocessing
3. Logistic regression modeling
4. Model evaluation using confusion matrix and recall

## 5. Model & Evaluation
A logistic regression model was used, prioritizing recall for churned customers to reduce missed churn cases.

## 6. Key Takeaways & Next Steps
- Contract type and tenure were strongly related to churn.
- This analysis can be extended to design targeted retention experiments.
